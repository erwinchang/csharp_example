# csharp_example

## StatusStrip工具列控制項

一般視窗應用程式會提供 狀態列在視窗的正下方
用來顯示系統的相關訊息

StatusStrip提供下例類別:
1.ToolStripStatusLabel: 顯示文子
2.ToolStripStatusBar: 建度表
3.ToolStripDropDownButton:下拉選單
4.ToolStripSplitButton
