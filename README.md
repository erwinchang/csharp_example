## ContextMenuStrip

用意: 讓物件點右鍵可以看到選單內容

在應用程式的輸入介面上按右鍵常會出現捷徑功能表，可以加快操作

程式設定如下
```
this.button1.ContextMenuStrip = this.contextMenuStrip1;
```
